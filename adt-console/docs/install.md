# ADT Operator Console Installation Guide

## Linux

### AppImage
1. Download `adt-console_0.1.0_amd64.AppImage` from the [releases page](https://github.com/human86x/adt-framework/releases).
2. Make it executable: `chmod +x adt-console_0.1.0_amd64.AppImage`
3. Run it: `./adt-console_0.1.0_amd64.AppImage`

### Debian/Ubuntu
1. Download `adt-console_0.1.0_amd64.deb`.
2. Install: `sudo dpkg -i adt-console_0.1.0_amd64.deb`

## macOS

### Homebrew
```bash
brew install human86x/tap/adt-console
```

### DMG
1. Download `adt-console_0.1.0_x64.dmg` (Intel) or `adt-console_0.1.0_aarch64.dmg` (Apple Silicon).
2. Open the DMG and drag ADT Console to your Applications folder.

## Windows

### Winget
```powershell
winget install ADTFramework.Console
```

### MSI
1. Download `adt-console_0.1.0_x64.msi`.
2. Run the installer and follow the prompts.
